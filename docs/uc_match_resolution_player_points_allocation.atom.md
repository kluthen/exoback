# Player Points Allocation Logic

## INTENT
Allow players to allocate Attribute Points to characters in their roster.

## THE RULE / LOGIC
The player allocates 1 Attribute Point to any character in their roster (HP, Attack, or Defense) — or defers it if thresholds allow.
